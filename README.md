**Dartboard** is a modern self-hostable Platform as a Service (PaaS) — a fork of Dokploy — that makes it easy to deploy and manage applications and databases on AWS Lightsail and other providers.

## ✨ Features

- **Applications**: Deploy any type of application (Node.js, PHP, Python, Go, Ruby, etc.).
- **Databases**: MySQL, PostgreSQL, MongoDB, MariaDB, libsql, Redis.
- **Backups**: Automated backups to external storage.
- **Docker Compose**: Full native support.
- **Multi Server / Multi Node**: Scale across Lightsail instances or any VPS.
- **Templates**: One-click open-source templates.
- **Traefik + Real-time Monitoring**: Built-in routing and observability.
- **CLI/API + Notifications**: Full automation.
- **AWS Lightsail Integration** (coming soon): Automated provisioning with fresh free-tier accounts.
